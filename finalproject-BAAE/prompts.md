# Prompts para Documentación del Asistente de Recetas Inteligente

## 1. Documentación General
```
Eres un experto en desarrollo de proyectos, conoces arquitectura de software y sabes de prompt engineering:
Estoy iniciando un proyecto el cual es un Asistente de Recetas Inteligente, necesito que me ayudes a generar la documentacion tecnica del proyecto
```
